Implementation Design — Adaptive Weather Effects

Style: subtle by default, intense for severe weather. Review each section and confirm.

⚡ 1. Lightning Screen Flash

Add a full-viewport white overlay on each strike. Severity scales with storm intensity — flicker at base, single clean flash per bolt.

✦ 2. Star Colour Variance

Assign spectral class on particle creation. Mix: 40% white, 25% pale blue (O/B), 20% yellow (F/G), 10% pale orange (K), 5% red (M giants). Brighter stars more likely to be blue.

Subtle — same alpha as now, just coloured

☀ 3. Solar / Lunar Halo

Thin radial gradient ring at 22° from sun/moon centre. Only visible when cloudDim ≥ 0.5 (clear enough sky). Intensity: haloAlpha = (cloudDim - 0.5) * 0.5 — fades out as clouds increase.

ConditionHalo
sunny / windyFull halo
partlycloudyPartial halo
cloudy / fog / rainNone
Subtle ring — not distracting

🌈 4. Rainbow Arc

Rendered after rainy or pouring conditions when sunElevation > 5°. Arc centred at antisolar azimuth (sun azimuth + 180°), radius 42° projected to canvas. 6-colour ROYGBIV gradient, 5px bands. Fades in after 3s, fades out if condition changes.

Visible but translucent — alpha ~0.25

❄ 5. Hexagonal Snowflakes

Replace ctx.arc with 6-arm crystal: main arms + 2 branches per arm at 50% length. Rotate each flake on its own axis as it falls. Larger flakes (r > 4) get branches; small ones stay simple arms only.

Same opacity as current snow particles

🌫 6. Improved Fog

Replace 12 flat ellipses with 4 stratified layer bands (y = 20%, 40%, 60%, 80% of canvas). Each layer drifts at a different speed. Add turbulence via Math.sin(x * freq + t * speed) to the y-offset of each blob. More blobs per layer (5 vs 3).

Same alpha range — just more organic shape

◎ 7. Bokeh Aperture Blades

After drawing the existing radial glow, add 6 faint lines from centre to edge at 60° intervals (hexagonal iris). Line alpha = bokeh.alpha * 0.35. No additional particles — just extra draw calls on existing bokeh circles.

Very subtle — barely visible on small bokeh

〜 8. Smoother Aurora

Replace per-pixel lineTo loop with quadraticCurveTo across 10 control points per band. Add vertical oscillation: yBase += Math.sin(t * 0.02 + bandIndex) * 8. Same colour/alpha as current.

Visual improvement + ~50% less canvas ops

Three targeted fixes — no visual change:
